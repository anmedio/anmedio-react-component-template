# Anmeido-react-component-template

🐊 React boilerplate for component build made by ANMEDIO [ANMEDIO][anmedio] with [NikolasMelui][nikolasmelui].

***

## Requirements

Operating system:

- Ubuntu/Debian
- CentOS/RHEL
- MacOS
- Windows

Node:

- NodeJS >= 10.x
- NPM >= 6.x

### Installation

```bash
npm i
```

### Quick start

```bash
npm run react:start
```

```bash
npm run gulp:start
```
